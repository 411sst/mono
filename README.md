# Meownopoly (MVP scaffold)

Server-authoritative multiplayer Monopoly-style prototype with:
- Quick Match queue
- Authoritative turn engine
- Versioned state updates + conflict rejection
- Map catalog/validation pipeline
- Mutation persistence to local file store
- SQL schema for PostgreSQL deployment
- Browser client with board/dashboard/actions

## Run

```bash
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test
```
